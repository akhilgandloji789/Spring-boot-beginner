# Spring Boot Beginner

A beginner-friendly Spring Boot project created as part of the Spring Boot Web Development Bootcamp by Being Infinity.

## 📖 Overview

This project demonstrates the fundamentals of Spring Boot web development, including:

- Spring Boot setup and project structure
- HTML, CSS, and JavaScript integration
- Static resources management
- REST API development
- User Management API
- Swagger/OpenAPI documentation
- Git and GitHub workflow

## 🚀 Features

- Responsive web pages using HTML and CSS
- JavaScript-based interactivity
- User Management REST API
- API documentation with Swagger UI
- Spring Boot backend application
- Git version control

## 🛠️ Technologies Used

- Java 21
- Spring Boot
- Maven
- HTML5
- CSS3
- JavaScript
- Git & GitHub
- Swagger/OpenAPI
